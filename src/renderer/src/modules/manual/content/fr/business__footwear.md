# Chaussures

Choisir **Chaussures** comme type d'entreprise active exactement le même ensemble de modules que Vêtements : le **suivi des variantes de taille/couleur**, **Retours**, et l'ensemble partagé des modules **Logistique**. Cela fonctionne de façon identique — voir le chapitre *Vêtements* pour la présentation complète de Gérer les variantes, comment une variante est sélectionnée dans Facturation, et comment fonctionnent les Retours. La seule vraie différence est cosmétique : les suggestions du champ de taille penchent vers le pointage des chaussures (5, 6, 6,5, 7 … jusqu'à 12) plutôt que les tailles de vêtements (XS–3XL), mais le champ de taille lui-même accepte n'importe quel texte dans les deux cas, afin que vous puissiez saisir la convention de tailles que vous utilisez réellement.

## Ce qui est partagé avec toute entreprise

Facturation, facturation, paiements, Clients, Produits, Rapports, Sauvegarde, et Utilisateurs & Permissions fonctionnent tous exactement comme décrit dans leurs propres chapitres.
