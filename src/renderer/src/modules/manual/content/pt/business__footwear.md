# Calçados

Escolher **Footwear** como seu tipo de negócio ativa exatamente o mesmo conjunto de módulos que Vestuário: **rastreamento de variações de tamanho/cor**, **Devoluções** e o conjunto de módulos compartilhado de **Logística**. Funciona de forma idêntica — veja o capítulo *Vestuário* para o passo a passo completo de Gerenciar Variações, como uma variação é selecionada no Faturamento e como as Devoluções funcionam. A única diferença real é estética: as sugestões do campo de tamanho tendem para numeração de calçados (5, 6, 6.5, 7 ... até 12) em vez de tamanhos de roupas (XS–3XL), mas o próprio campo de tamanho aceita qualquer texto de qualquer forma, para que você possa informar a convenção de numeração que realmente usa.

## O que é compartilhado com todo negócio

Faturamento, emissão de notas, pagamentos, Clientes, Produtos, Relatórios, Backup e Usuários e Permissões funcionam exatamente como descrito em seus próprios capítulos.
