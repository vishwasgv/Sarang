# Independent Consultant

## What's included

Independent Consultant is built on Sarang's shared service-business foundation — appointments, a service catalog, provider schedules, and the notification queue — plus four modules aimed at project- and retainer-based consulting work: **Leads**, **Service Projects**, **Retainers**, and **Time Tracking**.

## Leads

**Leads** is a drag-and-drop pipeline board (Open → Contacted → Proposal → Won → Lost). Each card carries contact details, source, an estimated deal value, and an assigned consultant. The KPI bar shows total leads, open leads, won leads, and total pipeline value.

## Service Projects

**Projects** tracks each client engagement — project name, type, stage, status (Active, On Hold, Completed, Cancelled), total contract value, start/expected-end dates, and the assigned consultant. Expand a project to manage:

- **Milestones** — a named deliverable with its own amount and status (Upcoming, In Progress, Completed, Invoiced, Paid). Once a milestone has an amount, a **Generate Invoice** button turns it into a real invoice.
- **Sprints** — optional short work cycles with a goal, date range, and status, useful for structuring longer engagements even outside a strict software context.

## Retainers

**Retainers** manages recurring client agreements — a title, a retainer type (Fixed Fee, Hourly Bucket, or Deliverable-Based), a monthly amount, and a billing day of the month. For an active retainer that hasn't been invoiced for the current month yet, a **Generate Invoice** action creates that month's invoice in one click; Sarang tracks which period was last invoiced so you don't double-bill.

## Time Tracking

**Time Tracking** ("Log Hours") records billable time against a project — date, staff member, description, hours, and rate per hour, with the amount calculated automatically. Entries default to the current month's view. Select one or more unbilled entries and use **Generate Invoice** to combine them into a single invoice; once an entry is billed it can no longer be edited or deleted. The KPI bar shows hours logged this month, unbilled hours, and unbilled amount.

## Language

Independent Consultant is one of Sarang's 24 dedicated service-business templates, and like nearly all of them its interface is **English only**, regardless of which language you've set elsewhere in Sarang.
