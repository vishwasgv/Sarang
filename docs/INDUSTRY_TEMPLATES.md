## TEMPLATE ACTIVATION MECHANISM

Templates are activated by setting `business_type` during onboarding. No separate installation required.

```typescript
// Industry template loads based on business_type from BUSINESS_PROFILE
const template = await industryTemplateService.loadTemplate(business.business_type);
// Activates relevant modules, dashboard layout, and navigation items
```

Feature flags (stored in `INDUSTRY_TEMPLATE_SETTINGS`) control:
- Which navigation items appear in sidebar
- Which dashboard widgets are shown
- Which report types are available
- Which roles are enabled (e.g., Kitchen Staff only for Restaurant)
