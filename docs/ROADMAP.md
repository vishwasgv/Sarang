## PHASE COMPLETION CRITERIA

Phase 0 complete when:
- Electron app launches without errors
- Prisma connects to SQLite successfully
- Authentication screens functional
- Core folder structure established

Phase 1 complete when:
- Full billing workflow functional (product → invoice → payment → receipt)
- Inventory auto-deducted on invoice creation
- All 4 industry templates (Restaurant, Retail, Hardware, Distributor) functional
- Reports generate correctly from source data
- Backup and restore both tested successfully
- All role-based permissions enforced

Phase 1.1 complete when:
- App startup consistently under 3 seconds
- Invoice creation under 1 second
- No frame drops during standard workflow navigation
- All table filters and search working correctly
- All user-facing documentation complete

## RELEASE VERSIONING GUIDE

| Version Pattern | Meaning |
|----------------|---------|
| vX.0.0 | New phase launch |
| vX.Y.0 | Feature additions within a phase |
| vX.Y.Z | Bug fixes and patches |

Never increment major version for bug fixes.

Never ship a vX.0.0 without completing the full phase criteria.
