## TYPOGRAPHY SPECIFICATION

### Primary Font
**Inter** — Google Fonts, hosted locally

Font weights used:
- 400 (Regular) — body text, descriptions
- 500 (Medium) — labels, button text, nav items
- 600 (SemiBold) — card headings, table headers
- 700 (Bold) — page titles, KPI values

### Font Stack
```css
font-family: 'Inter', system-ui, -apple-system, BlinkMacSystemFont, 
             'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;
```

### Font Size Scale

| Token | Size | Weight | Usage |
|-------|------|--------|-------|
| `text-3xl` | 30px | 700 | Page titles |
| `text-2xl` | 24px | 600 | Section headings, KPI values |
| `text-xl` | 20px | 600 | Card titles |
| `text-lg` | 18px | 500 | Sub-headings |
| `text-base` | 16px | 400 | Body text (primary) |
| `text-sm` | 14px | 400 | Body text (secondary), table cells |
| `text-xs` | 12px | 500 | Badges, labels, captions |

### Numeric Display Rule
All monetary values and quantities must use `tabular-nums` to prevent layout shift during updates:
```css
font-variant-numeric: tabular-nums;
/* Tailwind: class="tabular-nums" */
```

## BRAND CLEARSPACE RULE

The Aszurex logo and wordmark must always have clearspace equal to the cap-height of the "A" in "Aszurex" on all four sides. No text, icons, or UI elements may intrude into this clearspace.
