## EXECUTIVE SUMMARY

Sarang Business OS Lite is a free, offline-first Business Operating System built for small and medium businesses across the globe.

It is designed to help businesses manage billing, inventory, reporting, analytics, operations, customers, suppliers, expenses, and day-to-day workflows without recurring software costs, cloud dependency, or vendor lock-in.

The software will be:

- Free Forever
- Offline First
- Privacy First
- Ownership First
- Self-Service First
- Global First
- Easy to Install
- Easy to Operate
- Easy to Maintain

Sarang is not intended to become a SaaS platform.

Sarang is not intended to become a subscription business.

Sarang is not intended to become a cloud platform.

Sarang is not intended to become a payment processor.

Sarang is intended to become:

The most trusted offline-first Business Operating System for MSMEs worldwide.
